# Assets CIVICTWIN Semarang

Folder ini menyimpan aset visual untuk prototype CIVICTWIN.

## Struktur

```text
src/assets/
|-- brand/
|   |-- civictwin-logo.png
|-- README.md
```

## Aset Saat Ini

- `brand/civictwin-logo.png`: logo CIVICTWIN Semarang yang digunakan pada layout aplikasi.

## Catatan

- Aset visual bersifat bagian dari prototype lomba.
- Jika menambahkan gambar baru, gunakan nama file deskriptif dan simpan pada subfolder sesuai konteks.
- Jangan menaruh data pribadi, foto bukti laporan asli, atau dokumen sensitif di folder aset publik.
